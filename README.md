# MusicTune
A Music streaming web player like youtube music built on MERN stack.

It has functionality like specific lyrics, album details, artist details, searching using filters.

### Tech Stack Used

* React
* Tailwind CSS
* Firebase
* MongoDb
* Express
* Node.js
